@youneedtocode

@youneedtocode is a Twitter bot created using Zach Whalen's engine in Google Spreadsheets. It tweets to me (@alvernchen) every four hours to remind me of things I should be doing instead of checking Twitter.



Secret Santa

Since we're all too busy now to meet up before the gift exchange, this is a site I bring out for the winter holidays. Names are entered in a Python program and output into two text files - one sorted by alphabet, one randomized - that is used to match gift givers with gift receivers for Secret Santa.
