Projects

FE Tier List Maker

The FE Tier List Maker is a way for fans of the Fire Emblem series of games to sort and rank their favourite characters.

The functionality was built off of an existing tier list maker for Super Smash Bros. and extended to implement mobile-first responsive design, refactor the code to reuse it for other tools available on the site, and more. Check out the repository here.

Over the years, it's received pull requests from other contributors and currently averages over 400 clicks a month through Google Search alone.

You can save and load your lists as text or screenshot them to post elsewhere. It currently features over 1600 characters from 17 games, and I'm still working on new ways to search and filter them all.
